Rolling Sky 2D

How to install Rolling Sky 2D: (Note: when downloading from MediaFire, make sure to click off any ads!)

  1. Download Rolling Sky for Desktop here. (Download only if you don't have a copy already)
  2. Download the Rolling Sky 2D DLL here.
  3. Unzip the Rolling Sky for Desktop.
  4. Navigate to "Rolling Sky_Data > Managed > Assembly-CSharp".
  5. Replace the AssemblyCSharp.dll with the Rolling Sky 2D Dll.
  6. Run the EXE file and enjoy!
