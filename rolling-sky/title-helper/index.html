A Guide to Renaming Levels

I promise everything you need to know is on this page. Follow the instructions to the letter.

Setting Up

I recommend using Notepad++, it's the easiest to use in my opinion.
If you use Notepad++, select the following option shown in the image below.

How to Use

Type in or paste in level names separated by new lines.
Sometimes, the generator will output a {CR} or a {LF}, which you need to replace manually because copy and paste will just turn those into a whole different character.

In Notepad++, the characters can be replaced like this:

(Replace {CR} with \r and {LF} with \n, both using Extended mode)


Generator

Type your level names into the generator (Each line is a seperate name).
Note that some devices may not show special characters.

How to rename levels

First, find your level with CTRL+F, make sure its only your level and not a text containing your level.
Then, replace the level name with and character infront of it (Ex. In the case with Massif, replace the BS character) with the output from the generator.

Enjoy!

If the game is crashing, you did something wrong.
You should re-read the second line of "How to Use".
